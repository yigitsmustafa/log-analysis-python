# Log Analysis Project

## 🇬🇧 English

This project is a simple log analysis tool developed using Python.

### Features
- Counts ERROR and WARNING logs
- Detects the most frequent error
- Identifies potential suspicious activity

### Technologies
- Python

### Purpose
This project was created to practice basic log analysis concepts related to cybersecurity and digital forensics.

---

## 🇹🇷 Türkçe

Bu proje, Python kullanılarak geliştirilmiş basit bir log analiz aracıdır.

### Özellikler
- ERROR ve WARNING kayıtlarını sayar
- En sık tekrar eden hatayı tespit eder
- Olası şüpheli aktiviteleri belirler

### Kullanılan Teknolojiler
- Python

### Amaç
Bu proje, siber güvenlik ve adli bilişim alanlarında log analizi mantığını öğrenmek amacıyla geliştirilmiştir.
